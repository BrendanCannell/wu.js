Basics

Iterators represent a stream of data. To get the next value in the stream, you call the iterator's next method. This enables both lazy and infinite sequences. Most of the time you don't need to call next yourself: when you use a for-of loop, you're using iterators behind the scenes.

Anything can create iterators — they just need to make an object with the proper next interface — but generator functions and the yield expression provide extra convenience.

{% highlight js %} // Yield every positive integer. function* positiveIntegers() { let i = 1; while (true) { yield i++; } } // Log some of them. for (let x of positiveIntegers()) { if (x > 5) { break; } console.log(x); } {% endhighlight %}
