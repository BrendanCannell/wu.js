##### `wu.has(iterable, thing)`
##### `wu(iterable).has(thing)`

Returns `true` if `thing` is in the iterable (using `===` comparison), otherwise
returns `false`.

{% highlight js %}
wu(["uno", "dos", "tres"]).has(1);
// false
wu.count().has(5);
// true
{% endhighlight %}
